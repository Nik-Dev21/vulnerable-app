# VibePay - AI-Generated Payment Dashboard

A quick payment processing dashboard built with Cursor + Bolt. Ships fast, handles payments, manages users.

## Features
- User authentication with session tokens
- Payment processing via Stripe
- Admin dashboard with user management
- File upload for receipts
- Search across transactions

## Stack
- Node.js + Express
- Python Flask microservice
- SQLite database
- React frontend

## Setup
```bash
npm install
node server.js
```

Built in 2 hours with AI. Ready for production!
